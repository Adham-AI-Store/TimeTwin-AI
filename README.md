## Overview

**TimeTwin AI** is an AI-powered productivity assistant designed to help users plan their day, organize tasks, create study plans, and improve personal productivity through intelligent conversations.

The project focuses on turning AI into a practical daily tool rather than just a chatbot.  
It is being built as an MVP with a backend-first architecture using **Python + FastAPI**, combined with a modern frontend and LLM-powered interactions.

---

## Problem

Many students and self-learners struggle with:
- organizing study schedules
- breaking goals into daily tasks
- staying consistent with productivity habits
- managing tasks and plans in one place

TimeTwin AI aims to solve this by acting as a smart productivity companion that can help users plan, organize, and stay on track.
